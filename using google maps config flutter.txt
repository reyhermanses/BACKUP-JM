using google maps

1. create map key di google console
2. register nama android cth : com.example.member_registration
3. jalankan hash, sha1 di copy kembali ke formfield setelah register android sdk
4. androidmanifest.xml di tambahkan
   - <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>
    <uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
   - <meta-data android:name="com.google.android.geo.API_KEY"
           android:value="YOUR KEY HERE"/>
